Arrays in General

Arrays in java are useful for storing similar pieces of data.
If you ever need to keep track of anything that can be arranged into a list, you should use an array.

For example, if you had a list of grades in a class, the grades could all be stored in an array.
The array would be initialized, then filled with the students' grades.

Arrays can also be used to store more than a list. If you have data that can be arranged into a matrix, you can use a 2-dimensional array instead of a 1-dimensional array.
So, if a 1D array is a list of items, a 2D array is a list of lists of items.

...and a 3D array would be a list of lists of lists of items,
and a 4D array would be a list of lists of lists of lists of items...

1D Arrays

A 1 dimensional array can be used to store a list of items or objects. The example on the previous page was a list of grades.

This section will go over how to create, access, modify, and sort 1-dimensional arrays.

Creating a 1-dimensional array

An array initializer is used to create an array. It can either create an empty array with a certain number of empty spaces in it, or it can create an array that is already filled with items.

int[] myArray = new int[10];
creates an empty array that can be filled with integers.

int[] myPrefilledArray = {1, 2, 3, 4, 5};
creates an array that already contains some integers.

Reading things from a 1-dimensional array

To be able to do anything with the data stored in the arrays, you must remember that arrays are divided into sections. To access a single item in the array, you must specify which of the items you want to get.

However, to get the first item in the array is actually at the array index (column) of 0.
Each item is at its number, minus 1.
So, in the example to the right, 1 has an index of 0, and 5 has an index of 4.

Index01234
Value12345

To actually retrieve the int from the index in the array, you use
int myNum = myPrefArray[index];
where index is the index in the array.

If you ask for the data at an index in the array that does not exist, you will get an ArrayIndexOutOfBoundsException exception. One way to prevent this is by making sure that you never ask for an index that does not exist.
