Arrays in General

Arrays in java are useful for storing similar pieces of data.
If you ever need to keep track of anything that can be arranged into a list, you should use an array.

For example, if you had a list of grades in a class, the grades could all be stored in an array.
The array would be initialized, then filled with the students' grades.

Arrays can also be used to store more than a list. If you have data that can be arranged into a matrix, you can use a 2-dimensional array instead of a 1-dimensional array.
So, if a 1D array is a list of items, a 2D array is a list of lists of items.

1D Arrays

A 1 dimensional array can be used to store a list of items or objects.

This section will go over how to create, access, modify, and sort 1-dimensional arrays.

Creating a 1-dimensional array

An array initializer is used to create an array. It can either create an empty array with a certain number of empty spaces in it, or it can create an array that is already filled with items.

int[] myArray = new int[10];
creates an empty array that can be filled with integers.

int[] myPrefilledArray = {1, 2, 3, 4, 5};
creates an array that already contains some integers.

Arrays can be created to contain any data type - int, double, String, or Object., but for these examples ints are simplest.

Reading things from a 1-dimensional array

To be able to do anything with the data stored in the arrays, you must remember that arrays are divided into sections. To access a single item in the array, you must specify which of the items you want to get.

However, to get the first item in the array is actually at the array index (column) of 0.
Each item is at its number, minus 1.
So, in the example to the right, 1 has an index of 0, and 5 has an index of 4.

To actually retrieve the int from the index in the array, you use
int myNum = myPrefArray[index];
where index is the index in the array.

If you ask for the data at an index in the array that does not exist, you will get an ArrayIndexOutOfBoundsException. One way to prevent this is by making sure that you never ask for an index that does not exist.

Modifying a 1-dimensional array

The contents of arrays can be changed in a way similar to how they are accessed.

Just as
myInt = myArray[index]
retrieves the value stored in the specified index,

myArray[index] = myInt
will set the value stored at that index to whatever myInt was.



Now that we can change the contents of an array, we can also set the contents of an empty array.
int[] myEmptyArray = new int[10] creates an empty int array with 10 empty spaces.

Each of those spaces can have its value set to an int. This is useful when you do not know what the contents of the array will be when writing the program, and need to set them programmatically later.

Sorting Arrays

There are 2 ways to sort arrays - selection sorting and insertion sorting.

Selection Sorting

A selection sort algorithm assumes a sublist that covers the entire array. It passes through each of the indeces in the sublist, comparing each value to each other to find the smallest value. When it has found the smallest value, it exchanges it with the first value in the sublist. It then moves the sublist boundaries one index to the right, and repeats the cycle until it reaches the end of the list.

The last 2 lines do not change because the last 2 numbers in the list were already in order. However the algorith does not know that and continues until it has gone over each index.

Insertion Sorting

An insertion sort algorithm gives the same end result as the selection sort, but does so in a different way. It chooses the first element in the array and finds its place in the array, moving any elements that are in the way.
It is less effcient than the selection sort in large lists, because it has to move more of the elements over.

Searching through arrays

Searching through arrays looking for a certain value can be done with loops. Either a while loop or a for loop will work.





while()

In the while loop example, we set an int to act as the index of the array. Then, we make the loop continue while the index is less than the length of the array.
Each time the loop runs, it checks the value of the index against the target value and does something if they are equal. Then it increments the index by 1.




for()

In the for loop example, the index is created and incremented in the loop parameters along with the condition it needs to meet. The condition is still index < myArray.length
The for loop puts everything in the while loop together into the loop parameters, making it perfect for iterating through arrays.

A for loop can also be used to iterate through every index in the array. In this case, a foreach loop can be used.

2D arrays

To store a matrix/grid of items instead of a list, you can make a list list.

Making a 2D array

The array initializer for a 2D array is similar to that of a 1D array.
1d init: int[] myArray = new int[10]
2d init: int[][] myArray = new int[10][10]

A prefilled 2d array would be created in the same way as the 1d one, but it would be an array of arrays. It takes the form of a 1-dimensional array that, instead of having an int at each index, has another array at each index. Each index of the array inside the array would then contain an int.
