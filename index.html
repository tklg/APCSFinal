Project proposal

To make a webpage that details what arrays are and how they work.

  • show examples of arrays
  • show examples of uses of arrays
  • explain all examples
  • have small inline demos alongside the examples that allow the examples to run
  • demos will use javascript but will do the same thing

The page will be separated into "slides" - will be able to scroll through all pages, but will also be able to focus onto a single page
