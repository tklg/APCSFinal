Project proposal

Theodore Kluge

Objective

To make a webpage that details what arrays are and how they work.

  • show examples of arrays
  • show examples of uses of arrays
  • explain all examples
  • have small inline demos alongside the examples that allow the examples to run
  • demos will use javascript but will do the same thing as their java equivalents

Content

The page will be separated into "slides" - will be able to scroll through all pages, but will also be able to focus onto a single page. The user will be able to select a subtopic from a navbar on the side or top.

Each page will contain a single subtopic of arrays and how it can be used, along with an example of the topic.

Subtopics

  • 1-dimensional arrays
    • Creating
    • Accessing
    • Modifying
    • Sorting
    • Searching
  • 2-dimensional arrays
    • Creating
    • Accessing
    • Modifying
    • Searching
  • Iterating through arrays
    • for
    • while
  • Arraylists
    • Creating
    • Accessing
    • Modifying
    • Sorting
    • Searching
